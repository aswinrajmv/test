This is a Python-based automated test program for performing load transient testing on a Power Distribution Network (PDN).
The script uses PyVISA to communicate with laboratory instruments and automates the testing process.

The program performs the following tasks:
    Initializes and configures instruments
    Applies load transient conditions
    Measures voltage behavior of power rails
    Captures waveform data from the oscilloscope
    Logs results to a CSV file
    Evaluates Pass/Fail conditions
    Calculates simple statistics
    Generates a test report
This project was developed as part of a hardware validation assignment.
